# Is GitHub Up?

A simple Next.js app that checks GitHub's live status and shows:

- `Yes` for normal operation
- `Kinda` for partial/minor issues
- `No` for major/critical outages

Status data comes from:

- `https://www.githubstatus.com/api/v2/status.json`

## Run locally

```bash
npm run dev
```

Open `http://localhost:3000`.
